Platform Features

Everything in one platform

Google OAuth Login

Customers sign in securely with their Google account. No passwords — fast and familiar.

Instant Transfers

Deposit, withdraw, and transfer funds between accounts in real time with full history.

Banking Analytics

Real-time charts with NumPy & Pandas showing daily, weekly, and monthly trends.

Fraud Detection

Rules engine auto-flags suspicious transactions and alerts the manager in real-time.

Report Exports

Export full analytics reports as PDF, Excel, or CSV with a single click.

Role-Based Access

Customers and Managers have separate dashboards with 403 Forbidden protection.

Security Alerts

Email notifications for new logins, account lockouts, and suspicious activity.

Login History

Full login tracking with device, browser, and location — visible in Security tab.

Account Lockout

Auto-locks after 3 failed attempts with instant email alerts to the manager.

Setup Guide

Run locally in 4 steps

Quick Start

1

Clone the repository

git clone https://github.com/prithinga-devi/secure_bank.git
2

Install dependencies

pip install flask pandas numpy openpyxl reportlab werkzeug
3

Initialize the database

python init_db.py
4

Run the application

python app.py

Then open http://127.0.0.1:5000 in your browser
